## 🚀 Как использовать

### Вариант 1: Локальный сервер (OpenServer, XAMPP, MAMP)

1. Скачайте или склонируйте репозиторий
2. Поместите папку с работами в директорию `domains` вашего локального сервера
3. Запустите сервер
4. Откройте в браузере: `http://название-папки`

### Вариант 2: Хостинг с поддержкой PHP

1. Загрузите все файлы на хостинг
2. Перейдите по соответствующему URL

### Вариант 3: Просмотр без PHP (только дизайн)

Если у вас нет доступа к PHP-серверу, вы можете просто открыть HTML-файлы — вы увидите красивое оформление и примеры результатов (но PHP-код выполняться не будет).
