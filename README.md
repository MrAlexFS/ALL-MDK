## 🚀 Как использовать

### Вариант 1: Локальный сервер (OpenServer, XAMPP, MAMP)

1. Скачайте или склонируйте репозиторий
2. Поместите файлы в директорию `domains` вашего локального сервера
3. Запустите сервер
4. Откройте в браузере: `http://название-папки/имя_файла.php`

### Вариант 2: Хостинг с поддержкой PHP

1. Загрузите все файлы на хостинг
2. Перейдите по соответствующему URL

### Вариант 3: Просмотр без PHP (только дизайн)

Если у вас нет доступа к PHP-серверу, вы можете просто открыть HTML-файлы — вы увидите красивое оформление и примеры результатов (но PHP-код выполняться не будет).

## ⚙️ Настройка для работ с базами данных (№27-29)

Для корректной работы работ, использующих базы данных, необходимо:

1. **Запустить MySQL модуль** в OpenServer (Настройки → Модули → выбрать MySQL)
2. **Убедиться**, что в настройках OpenServer выбран MySQL (например, MySQL-5.7 или MySQL-8.0)
3. **Базы данных создаются автоматически** при первом запуске скриптов
4. **Параметры подключения** по умолчанию:
   - Хост: `localhost`
   - Пользователь: `root`
   - Пароль: `(пустой)`
