# Практическая работа №18: Работа с Pytest

**Вариант:** Система оценивания (GradeCalculator)

## Структура проекта
- `src/grade_calculator.py` – класс калькулятора оценок
- `tests/conftest.py` – фикстуры и фабрики
- `tests/test_grade_calculator.py` – тесты

## Запуск тестов
```bash
pip install -r requirements.txt
pytest
```
